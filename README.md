# MultiPHarser versie 0.2
De nieuwe UBB-parser, oftewel MultiPHarser. Speciaal beschikbaar gesteld voor PHPhulp.nl

Installatie:
* Voer 'composer install' uit om de /vendor-map aan te maken met het vanilla/nbbc project.
* You're ready to go! In /tests staat een test-script (Parsertest.php). Dit wordt nog verder uitgewerkt naar een goed voorbeeld.
